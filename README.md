## 🎮 What is this?
An interactive web-based music quiz where players listen to short audio clips and guess the song. Choose your genre — Rap or Pop — enter your name, and see how many you can get right before the timer runs out!

## ✨ Features
- 🎵 Short audio clip playback for each question
- 🎤 Genre selection — Rap & Pop
- ⏱️ Timer per question
- 🏆 Score tracking
- 📱 Responsive UI
- 🌌 Starry night background design
- 🔇 No third-party API — fully self-hosted audio

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Audio: Self-hosted local files

## 🚀 Getting Started
1. Clone the repo
```bash
   git clone https://github.com/devanshs-dev/music-recognition-app.git
   cd music-recognition-app
```
2. Install dependencies
```bash
   npm install
```
3. Start the server
```bash
   node backend/server.js
```
4. Open in browser
